## 문제
- 정수 배열에 연산을 하기 위해 만든 언어
- R: 배열에 있는 수의 순서를 뒤집음
- D: 첫 번째 수를 버림 - 배열이 비어있으면 에러
- 함수를 조합해서 사용할 수 있음
- 최종 결과를 계산

## 풀이
- 수행할 함수 P
- 배열의 개수 n
- 배열

- D의 개수가 n보다 클 경우 error 반환
- ArrayDeque에 넣은 뒤 R이 하나 나오면 뒤에서, 이후 하나가 더 나오면 앞에서 이런식으로 삭제
- 현재 r의 상태를 계속 갱신시키고 뒤집힌 상태라면 뒤에서부터 빼서 출력, 아니면 앞에서부터 빼서 출력
- queue에서 하나씩 빼면서 StringBuilder에 넣음 (마찬가지로 짝수면 앞에서, 홀수면 뒤에서)
