## 문제
- 공기청정기는 항상 1번 열에 설치되어있고 크기는 두 행
- 1초 동안
  - **미세먼지 확산**:
  - 상하좌우로 확산
  - 칸이 없거나 공기청정기가 있으면 X
  - 현재 미세먼지양 / 5로 확산
  - 현재 칸에는 (현재 미세먼지 - 확산 미세먼지 * 확산 칸) 만큼의 미세먼지가 남음
  - **공기청정기 작동**:
  - 위쪽 공기청정기는 반시계 방향, 아래쪽 공기청정기는 시계 방향으로 순환
  - 바람의 방향대로 미세먼지 한 칸씩 이동
- T초가 지난 후 구사과 방에 남아있는 미세먼지 양 출력

## 풀이
- 미세먼지 확산: queue에 0이 아닌 좌표 모두 저장해두고 하나씩 빼면서 상하좌우 미세먼지 더하기
- 공기청정기:
- 위쪽: 오른쪽 -> 위쪽 -> 왼쪽 -> 아래쪽으로 한바퀴 돌면서 0이 아닌 값이 있다면 현재 방향대로 한칸 이동시키기
- 아래쪽: 오른쪽 -> 아래쪽 -> 왼쪽 -> 위쪽으로 돌면서 먼지 이동시키기

## 개선점
- Queue를 이용할 필요가 없음
- 확산 -> for문으로 돌면서 temp에 바뀐값 표시해두고 마지막에 arr 교체
- 공기청정기 -> 바람이 부는 방향이 아니라 공기청정기가 먼지를 땡겨온다고 생각하고 풀이
