## 문제
- 다리에는 트럭이 최대 bridge_length대 올라갈 수 있다.
- 다리는 weight 이하까지의 무게를 견딜 수 있다.
- 최소 몇 초가 걸리는지 return
- 트럭은 1초에 다리 길이 1씩 전진
- 트럭은 1초에 한대씩 올라갈 수 있다.

## 풀이
- for문 안에 while문으로 돌면서 확인


## 복습
- Queue -> 다리
- 다리가 비어있다면 트럭을 하나 추가하고 다리위의 무게에도 트럭 무게를 올린다.
- 트럭이 하나 올라갔으므로 1초 추가
- 만약 다리가 비어있지 않은 경우 다리에 올라간 트럭의 개수가 다리의 길이와 동일하다면 (다 올라가있으면) 1초 뒤에는 가장 먼저 올라간 트럭이 다리를 지나가기 때문에 다리(큐)에서 제거하고 전체 무게에서도 해당 트럭의 무게를 제거 (시간은 트럭을 올릴때만 증가시킨다)
- 현재 다리 위 트럭의 무게와 올려야되는 트럭의 무게를 합했을 때 기준 무게보다 크다면 0을 넣고 시간을 증가시킨다. (큐의 길이가 다리의 길이와 같을때만 트럭을 뺄 수 있기 때문에 0을 넣는다. -> 시간의 흐름을 나타낸다고 생각)
- 어떠한 경우에도 포함되지 않는 경우 트럭을 다리에 올리고 시간과 전체 무게를 증가시킨다.
- 마지막으로 올라간 트럭이 다리를 지나가는 시간까지 합해서 리턴한다.
