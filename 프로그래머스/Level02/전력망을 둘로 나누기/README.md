## 문제
- 전선들 중 하나를 끊어서 전력망 네트워크를 2개로 분할
- 송전탑의 개수를 최대한 비슷하게 맞추고자 한다.
- n : 송전탑의 개수
- wires : 전선 정보

## 풀이
- 한 개씩 끊으면서 개수 확인
- nodeList에 전선 정보를 넣고 한개씩 지우면서 bfs 돌리기



# 복습
## 문제
- 전선들 중 하나를 끊어서 송전탑 개수가 가능한 비슷하도록 두 전력망으로 나누었을 때, 두 전력망이 가지고 있는 송전탑 개수의 차이(절대값)를 리턴

## 풀이
- 전선 하나씩 끊으면서 임의의 지점에서 bfs 탐색
