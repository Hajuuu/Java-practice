## 문제
- 큐에 대기중인 프로세스 중 우선순위가 더 높은 프로세스가 있다면 방금 꺼낸 프로세스를 큐에 넣는다.
- 해당 프로세스가 몇 번째로 실행되는지 확인

## 풀이
- 우선순위를 얻기 위해 우선순위큐에 정렬시킴
- priorities와 비교하여 현재 값과 우선순위 큐의 맨 앞 값이 같다면 큐에서 삭제하고 answer더하기


## 복습
- 큐에서 하나씩 꺼냈을 때 큐 안에 현재 꺼낸 숫자보다 큰 숫자가 있다면 다시 넣음
- 해야되는거 1. 우선순위 높은게 있는지 2. location에 저장된 프로세스인지
- 우선순위 큐에 priorities의 값을 모두 넣는다.
- priorities 돌면서 현재 값이랑 우선순위큐 값이 같다면 우선순위큐에서 제거하고 answer 증가
- 이때 index가 찾고자하는 값이면 answer에 1 증가시킨 뒤 리턴
- 우선순위 큐 내림차순 정렬 : `PriorityQueue<Integer> queue = new PrioirtyQueue<>(Collections.reverseOrder());`
- priorities를 처음부터 끝까지 순회하면서 답이 안나왔다면 현재까지의 answer에 찾고자하는 index값 더하기
