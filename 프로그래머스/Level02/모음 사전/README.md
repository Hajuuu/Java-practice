## 문제
- word가 매개변수로 주어질 때 이 단어가 사전에서 몇 번째 단어인지 리턴
- 사전은 길이 5이하의 단어로 이루어져있음

## 풀이
- dfs 이용해서 풀이
- 사전 순서대로 모든 경우의수 만들면서 count 증가
- 현재 단어가 찾는 단어와 같다면 리턴
- A -> AA -> AAA -> AAAA -> AAAAA -> AAAAE 이 순서로 증가하기 때문에 dfs하나씩 호출하면서 count증가시키면 됨
- for문 위에서 5글자 이상의 단어를 만들지 않도록 if문으로 리턴하고 만약 단어를 찾았다면 count 증가하지 않도록 체크해둔다
