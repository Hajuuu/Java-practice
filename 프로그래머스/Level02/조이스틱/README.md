## 문제
- 알파벳 이름을 완성하는 문제
- 처음에는 "AAA" 이렇게 되어있음
- 조이스틱을 움직여서 원하는 이름을 만들고 조작 횟수의 최솟값 리턴

## 풀이
- 조이스틱 위아래 : A에서 현재 문자로 가는거랑 A -> Z -> 현재 문자로 가는 것중 최솟값 더하기
- 조이스틱 좌우 : A문자만 있는 구간은 갈 필요가 없음, 그 A구간 전까지 갔다가 다시 돌아와서 왼쪽 스틱을 움직이거나, 왼쪽 스틱을 움직인뒤 다시 오른쪽 스틱으로 넘어와서 남은 문자 바꾸는 방법 뿐
- 따라서 직진하는 경우 (n - 1)과 A구간을 기준으로 왼쪽먼저 처리, 오른쪽 먼저 처리의 3가지 경우를 비교해서 최솟값으로 더한다.
