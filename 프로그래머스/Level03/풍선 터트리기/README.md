## 문제
- 일렬로 나열된 n개의 풍선
  - 임의의 인접한 두 풍선을 고른 뒤, 두 풍선 중 하나를 터트린다.
  - 터진 풍선으로 인해 풍선들 사이에 빈 공간이 생겼다면, 빈 공간이 없도록 풍선들을 중앙으로 밀착시킨다.
- 인접한 두 풍선 중에서 번호가 더 작은 풍선을 터트리는 행위는 최대 1번만 가능
- 이후에는 번호가 더 큰 풍선만 터트릴 수 있음
- 1개 남을때까지 터트렸을 때 최후까지 남는 것이 가능한 풍선만 리턴

## 풀이
- 한번 외에는 번호가 더 큰 풍선만 터트릴 수 있으므로 양 옆의 숫자가 자신보다 작은 경우에는 최후까지 남을 수 없음
- 각 풍선의 왼쪽, 오른쪽 풍선의 최소값을 찾고 해당 값보다 현재 풍선의 번호가 작지 않도록 하면 된다.
