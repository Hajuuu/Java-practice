## 문제
- N과 사칙연산만 사용해서 number를 만들 때 N 사용횟수의 최솟값을 구하는 문제
- 사용횟수가 8보다 크면 -1 리턴
## 풀이
- DP로 풀이
- dp[i] -> N을 i개 사용했을 때 생기는 숫자들
- List<Set<Integer>> dp로 두고 여기에 N이 i개일때 생길 수 있는 숫자들을 저장한다. (중복인 경우가 생길 수 있으므로 Set에 저장함)
- 1개 -> 자기자신 저장
- 2개부터 8개 까지 저장
  - 2개: 1, 1
  - 3개: 1, 2 / 2, 1
  - 4개: 1, 3 / 3, 1 / 2, 2
  - 이런식으로 경우의수가 생기므로 이중 for문 이용해서 사칙연산해줌
  - 2개부터 8개까지 N을 연속해서 붙이는 경우도 있으므로 함께 더해줌 (repeat이용)
- 각 dp를 만든뒤 set안에 number가 있다면 i를 리턴
