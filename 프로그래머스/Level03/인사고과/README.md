## 문제
- 어떤 사원이 다른 임의의 사원보다 두 점수가 모두 낮은 경우, 그 사원은 인센티브를 받지 못함
- 점수의 합이 같으면 동석차, 동석차의 수만큼 다음 석차는 건너뜀
- scores : 각 사원의 근무 태도 점수와 동료 평가 점수 목록
- scores[0] : 완호의 점수
- 완호의 석차 리턴

## 풀이
- scores를 근무 태도 점수를 기준으로 내림차순 정렬, 근무 태도 점수가 같다면 동료 평가 점수를 오름차순 정렬
- 이후 scores를 순회하면서 현재까지 나온 동료 평가 점수의 최대값보다 작다면 제외 (근무 태도와 동료 평가 점수 모두 낮다는 의미이므로), 제외되는 사람이 완호면 -1 리턴
- 이때 모든 순위를 구하는게 아니라 완호의 점수를 따로 빼놓고 인센티브를 받을 수 있는 사람이라면 완호와 비교
- 완호보다 크면 answer 증가
