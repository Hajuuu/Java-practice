## 문제
- 장르 별로 가장 많이 재생된 노래를 두 개식 모아 베스트 앨범 출시
- 노래는 고유 번호로 구분
- 속한 노래가 많이 재생된 장르를 먼저 수록
- 장르 내에서 많이 재생된 노래를 먼저 수록
- 재생 횟수가 같은 노래 중에서는 고유 번호가 낮은 노래를 먼저 수록

## 풀이
- 각 장르별 재생회수를 구한다. -> 재생회수로 내림차순 정렬
- 장르별 노래의 번호와 재생회수를 저장해둔다. -> map value값에 list로 저장
- 장르별 노래의 재생회수와 고유 번호를 기준으로 정렬해둔다.
- 장르별 재생회수 정렬에 맞게 결과값 구해서 리턴 (노래가 두 개 이상이면 두개만, 한개면 하나만 넣어서 리턴)
