## 문제
- 조각은 한 번에 하나씩 채워 넣음
- 조각을 회전시킬 수 있음
- 조각을 뒤집을 수 없음
- 새로 채워 넣은 퍼즐 조각과 인접한 칸이 비어있으면 안됨

## 풀이
- game_board의 빈칸 모양과 완전히 일치하는 블록만 놓을 수 있음
- 블록을 (0,0)에 붙여놓는다고 생각하고 저장?
- 회전의 경우
- 파란색 블록 (0, 0) (1, 0) -> (0, 0) (0, 1)
- 보라색 블록 (0, 1) (1, 0) (1, 1) (1, 2) -> (1, 0) (1, 1) (1, 2) (0, 1)
- x y 를 뒤집은 결과만 블록으로 존재할 수 있음
