## 문제
- n : 선수의 수
- results: 경기 결과
- [A, B] : A 선수가 B 선수를 이겼다는 의미
- 정확히 순위를 매길 수 있는 선수의 수를 구하는 문제


## 풀이
- 플로이드 워셜 문제
- dp[i][j]의 2차원 배열을 만들고 이겼다면 1, 졌다면 -1로 값을 채운다.
- 예를 들어 [1, 4]면 dp[1][4] = 1, dp[4][1] = -1
- 3중 for문 돌면서 다시 한번 채워준다. 예를 들어 1 > 2, 2 > 5면 1 > 5이기 때문
  - i , j , k 로 돌면서 [i][k], [k][j] -> [i][j] 구한다.
- 값을 다 적어준 뒤 i를 돌면서 0이 아닌 값이 4개 이상인 경우 count (4개 이상 0이 아니어야 비교가 되는 수이기 때문)
